# Sistema de Gestión de Empleados

## 📋 Descripción
Aplicación Java simple para gestionar empleados de una empresa. Desarrollada como parte de la Actividad Formativa 2.

## 🚀 Funcionalidades
- ✅ Agregar empleados
- ✅ Buscar empleados por ID
- ✅ Buscar empleados por puesto
- ✅ Modificar datos de empleados
- ✅ Aplicar aumentos de salario
- ✅ Eliminar empleados
- ✅ Listar todos los empleados

## 🛠️ Tecnologías
- Java 17
- IntelliJ IDEA
